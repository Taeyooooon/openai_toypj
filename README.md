# How To Start
```bash
git clone https://github.com/Taeyooooon/openai_toypj.git
cd openai_toypj
npm install
npm start
```

```
// .env
REACT_APP_OPENAI_API_KEY=YOUR_OPENAI_API_KEY
```

# Deployed with Vercel
